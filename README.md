# 🎮 Tic-Tac-Toe Game

A simple **console-based Tic-Tac-Toe game** built in **C++** using Object-Oriented Programming concepts. The game supports two players, detects winners and draws, and allows replaying after each match.

## Features

* Two-player gameplay
* Win & draw detection
* Colored console interface
* Replay option

##  Technologies

* C++
* Object-Oriented Programming (OOP)
* STL (`vector`, `string`)

##  Run

```bash
g++ "Tic.c++" -o Tic
./Tic
```
